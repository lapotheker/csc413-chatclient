This chat client provides a complete messaging solution with server selection, user authentication, and real-time chat room functionality. The application uses JavaFX for its user interface and implements a scene-based architecture for seamless navigation between different application states.
Features

User Authentication: Secure login interface for user access
Server Selection: Choose from multiple chat servers
Chat Room Interface: Real-time messaging with other users
Theming Support: Multiple CSS themes 
Modular Architecture: Clean separation of concerns with scene-based design
